Proyecto 1 - Prisma CP

Este software fue diseñado y desarrollado específicamente para optimizar la eficiencia y la organización de los procesos clave de una empresa de imprenta. Utilizando Java como lenguaje principal en el backend y tecnologías frontend como HTML, CSS y Bootstrap para una interfaz intuitiva, Prisma - CP implementa un sistema de autenticación seguro con acceso diferenciado para roles de administrador, ejecutivo de cuenta, diseñador y operario, optimizando las tareas de cada área. Sus funcionalidades principales incluyen un control exhaustivo del inventario de materiales, la creación y visualización del estado de las órdenes de producción en tiempo real, una gestión integral de facturas de compra y venta, y un repositorio centralizado para el almacenamiento organizado de los diseños creados por el equipo de diseño, facilitando la colaboración y el flujo de trabajo.

Proyecto 2 - Ferreteria

Este proyecto tiene como objetivo simplificar y optimizar la administración de una ferretería a través de una aplicación web desarrollada con PHP y el framework CodeIgniter 3. Proporciona un acceso seguro mediante un sistema de autenticación por credenciales y ofrece herramientas esenciales como el control detallado del movimiento de materiales (entradas y salidas), la gestión de usuarios con la capacidad de crear nuevas cuentas, y un registro completo de los empleados. La implementación del método CRUD en cada módulo asegura una manipulación de datos eficiente y organizada. La interfaz, construida con Bootstrap, es intuitiva y adaptable a diferentes tamaños de pantalla gracias a su diseño responsive, mientras que la base de datos MySQL garantiza la integridad y disponibilidad de la información.
